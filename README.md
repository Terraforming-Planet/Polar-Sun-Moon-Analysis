# Polar Equinox & Lunar Standstill Analysis

Open-source scientific project for analyzing the apparent altitude of the Sun and Moon above the horizon during equinoxes at the North and South Poles using NASA JPL Horizons ephemerides.

## Objectives

- Retrieve official NASA JPL Horizons ephemerides.
- Analyze all equinoxes from 2006 to 2024.
- Compare Sun and Moon apparent altitude above the horizon.
- Analyze declinations.
- Investigate long-term behaviour over the 18.6-year lunar nodal cycle (major/minor lunar standstill).
- Produce reproducible tables, figures and statistical analysis.

## Data source

NASA JPL Horizons

https://ssd.jpl.nasa.gov/horizons/

## Output

- CSV
- Excel
- PDF report
- Scientific plots
- Statistical summary

This repository performs astronomical calculations only. It does not assume or prove any hypothesis in advance. Conclusions are derived solely from NASA ephemerides.
