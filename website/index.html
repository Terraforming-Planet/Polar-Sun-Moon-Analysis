Polar Sun Moon Analysis

Scientific Python project for analyzing seasonal apparent altitude and declination of the Sun and Moon at the North Pole and South Pole during vernal and autumnal equinoxes from 2006 through 2024.

Scientific scope

  • Official NASA JPL Horizons observer ephemerides only.
  • Sun and Moon apparent altitude above the local horizon.
  • Sun and Moon declination.
  • North Pole and South Pole topocentric observers.

NASA Horizons references

The project calls the machine-to-machine Horizons API programmatically with required query parameters. For browser use, open the Horizons web application. Developers should consult the official Horizons API documentation.

Outputs

Running python -m polar_equinox_analysis exports CSV, Excel, JSON, Markdown, HTML, PNG figures, plain-text summary, and PDF report files under outputs/.
